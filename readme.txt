仕様について

この電卓は通常の電卓のように使用できますが、以下の注意点があります。

・関数は必ずカッコ（）で閉じる。例：sin(90), log(5)
・角度の単位は度またはラジアンを必ず選択して下さい。また、式の記入中に変更しないで下さい。
　変えられますと、度とラジアンが混ざった計算式になってしまいます。
・本来、cos90°やsin180°は0になりますが浮動小数点数の計算誤差の影響で0となりません。
　とても小さい値が結果として帰ってくるため、お気をつけて下さい。
　例：cos90°= 6.123233995736766e-17
